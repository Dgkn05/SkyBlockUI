### Installation

- Download the latest or stable build. Builds can be found [here](https://github.com/TheRealKizu/SkyBlockUI/releases)
- Place it in your `plugins` folder.
- Restart your server.

- [x] Island Creation
- [x] Island Management
- [x] Help
- [x] Invite Management
- [x] Member Management
